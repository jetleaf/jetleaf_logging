jetleaf_logging library

üìù JetLeaf Logging API Entry Point

This library provides a unified import for JetLeaf's logging system, combining both the core logging API and all built-in printer implementations.

Instead of importing multiple modules, applications can access everything through a single entry point.

‚úÖ Includes

üì¶ Core Logging System

Re-exported from logging.dart:

  • log levels, types, and steps
  • Log, LogFactory, and configuration
  • listeners and log record models

Typical usage:

import 'package:jetleaf_logging/jetleaf_logging.dart';

final log = LogFactory.get('server');
log.info('Server started');

Re-exported from printers.dart:

  • pretty / flat / structured / hybrid printers

üéØ Intended Usage

Preferred single import for application developers:

import 'package:jetleaf_logging/jetleaf_logging.dart';
